# Cybersecurity Home Lab

## Overview
Built a virtual lab using Hyper-V to simulate a small network environment.

## Setup
- Windows VM
- Kali Linux VM
- Internal network configuration

## Tools Used
- Wireshark
- Nmap

## Activities Performed
- Network traffic capture and analysis
- Port scanning and service detection

## Key Learnings
- Identified open ports and associated risks
- Analysed packet structures and protocols
