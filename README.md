# Cybersecurity Home Lab

## Overview
This project simulates a small network environment using virtual machines to understand network behaviour and basic security testing.

## Lab Setup
- Windows 10 VM (target)
- Kali Linux VM (attacker)
- Configured on internal virtual network

## Tools Used
- Wireshark
- Nmap

## Activities Performed
- Verified connectivity between systems
- Performed port scanning using Nmap
- Captured and analysed network traffic using Wireshark

## Key Findings
- Identified open ports and running services
- Observed TCP/IP packet structure
- Analysed network traffic patterns

## Skills Gained
- Network analysis
- Basic penetration testing techniques
- Traffic inspection
